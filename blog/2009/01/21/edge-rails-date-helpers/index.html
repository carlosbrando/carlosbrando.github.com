Edge Rails: Date Helpers

Jan 21st, 2009

Uma nova opção foi adicionada aos date helpers do Rails. Métodos como o date_select, time_select, datetime_select, select_minute e outros receberam a opção :prompt. Com esta opção é possível definir um valor padrão como o primeiro elemento da lista. Veja um exemplo: 
<%= date_select("post", "written_on", :prompt => true) %>
O código acima devolverá os campos com as seguintes mensagens: [caption id=”attachment_2116” align=”aligncenter” width=”233” caption=”Exemplo com os valores padrão”][/caption] Também é possível personalizar a mensagem de cada campo: 
<%= date_select("post", "written_on",
      :prompt => {
        :day => 'Select day',
        :month => 'Select month',
        :year => 'Select year'
      }) %>
Com o código acima teremos algo assim: [caption id=”attachment_2117” align=”aligncenter” width=”319” caption=”Exemplo com valores personalizados”][/caption] Esta é uma opção muito bem-vinda, já que para conseguir o mesmo efeito antes era necessário um truquezinho.
Todos os exemplos dados aqui funcionarão somente no Ruby on Rails 2.3/3.0 ou superior. Você pode encontrar mais detalhes sobre esta e outras novidades acompanhando a série Edge Rails.
