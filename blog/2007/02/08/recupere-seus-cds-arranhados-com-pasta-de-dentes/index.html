Recupere Seus CDs Arranhados Com Pasta De Dentes

Feb 8th, 2007

Pode parecer esquisito, mas uma das maneiras de se salvar um CD riscado é passando pasta de dentes na sua superfície. Gabriel Torres do site clube do hardware ensina como fazer. Ele explica que em um CD os dados estão gravados numa camada metálica dentro dele, e esta camada é revestida por uma camada plástica transparente que serve para protegê-la.Quando esta camada plástica risca, o leitor não consegue ultrapassá-la para ler os dados gravados dentro do CD. Por isto a sugestão é polir a camada plástica usando pasta de dentes. Usando um chumaço de algodão, esfregue suavemente a pasta de dente sobre os ricos até removê-los. Depois lave com água e pronto! Se tiver sucesso, recomendamos fazer uma cópia do CD para evitar problemas no futuro. E então, fez o teste? Deixe um comentário para sabermos se deu certo. via Clube do Hardware
