Gerando Dados Para Testes Em Ruby

Dec 4th, 2007

Alguns já devem conhecer o Data:Faker do Perl, usado para gerar dados falsos como nomes, endereços, números de telefone, emails, etc.. A novidade é queÂ Ben CurtisÂ portou ele para um gem do Ruby. De acordo com Ben, a necessidade veio a tona quando ele precisou tirar screenshots de um novo software que havia criado e queria dados mais próximos da realidade.Achei o plugin muito interessante, principalmente para gerar dados para testes.Para instalar é fácil:
gem install faker
Usar é mais fácil ainda:
