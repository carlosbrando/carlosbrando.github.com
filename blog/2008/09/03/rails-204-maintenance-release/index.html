Rails 2.0.4: Maintenance Release

Sep 3rd, 2008

Se você ainda está usando a versão 2.0.2 do Rails isto é importante para você. O core team do Rails acabou de liberar uma versão de manutenção com a correção de vários bugs encontrados no Rails 2.0.2. Para atualizar seu Rails execute no terminal: 
gem install rails --version 2.0.4
A principal motivador desta versão foi a correção da vulnerabilidade de Denial of Service no REXML. Outras correções menores também foram incluídas. Fora estas correções não há nenhuma novidade importante nesta versão.
