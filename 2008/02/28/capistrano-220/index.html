Capistrano 2.2.0

28 de Fevereiro de 2008

Acabei de ver no blog oficial do Rails que saiu a versão 2.2.0 do Capistrano. O Capistrano (que também é o nome de um município no Ceará, será que tem algo a ver?) para quem não conhece é a ferramenta de deploy mais usada pelos Railers, que permite automatizar tarefas em vários servidores remotos ao mesmo tempo.

Para saber o que tem de novo, passe no blog oficial.

Eu estava esperando sair esta versão para ver se começo uma série “Edge Capistrano” aqui no blog. Vamos ver se dá certo, porque não conheço muito do código dele…
