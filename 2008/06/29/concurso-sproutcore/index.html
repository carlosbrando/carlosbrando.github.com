Concurso SproutCore

29 de Junho de 2008

O time de desenvolvimento do SproutCore está propondo um concurso que envolve criar aplicativos de demonstração usando o framework. O concurso vai até o dia 21 de julho.

Para participar, basta criar um aplicativo com cara de desktop usando o SproutCore e depois incluir o projeto no wiki do GitHub e enviar um email para sproutcore@sproutitmail.com.

O que você ganha? Uma camiseta e um link no site do projeto, ou seja, nada de mais… Mas para quem está tentando aprender a usar o framework talvez sirva de incentivo.
