Rubypub: Documentação De Gems Feita Por Você

Sep 9th, 2007

Caramba… mais um site lançado neste fim de semana. Agora é a vez do Rubypub, uma espécie de wikipedia de gems, módulos, classes e métodos em Ruby (e Rails).

O site já conta com milhares de gems cadastrados, só esperando uma boa alma escrever algumas linhas e deixar documentados os processos de instalação e uso deles.

O site ainda está bem no começo, o que significa que você vai encontrar muita coisa sem nenhuma documentação. Mas acredito que é apenas uma questão de tempo (se o site vingar, claro!). Mas como a comunidade Ruby on Rails é muito ativa, as chances de o pessoal começar a jogar informações no site é grande. Até porque o processo deveria ser um simples copiar e colar, já que é muito difícil encontrar algum plugin ou gem que não possua uma documentação junto com o código.
