Deixe as imagens do seu site mais bonitas usando flash

12 de Março de 2007

O swfIR (swf Image Replacement) promete acabar com as limitações de tratamento de imagens no seu site. Usando flash e javascript a ferramenta permite que você aplique efeitos visuais interessantes nas imagens.

Para usar é muito simples. Primeiro você deve baixar o swfir. São dois arquivos um .js (javascript) e outro .swf (flash), os dois devem ser colocador na mesma pasta no seu servidor web.

Agora é necessário editar a página onde as imagens receberão o tratamento. Entre as tags <head> e </head> inclua o seguinte código:

<script type="text/javascript" src="swfir.js">

Altere a propriedade src para o caminho onde você colocou os arquivos.

Logo abaixo, coloque o código para realizar o efeito desejado. Neste caso, bordas arredondadas:

<script type="text/javascript">

window.onload = function(){
round = new swfir();
round.specify('src', '/swfir.swf');
round.specify('border-color', 'ffffff');
round.specify('border-radius', '10');
round.swap("img");
}

</script>

Para ver outros efeitos possíveis clique aqui.
