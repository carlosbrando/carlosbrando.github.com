Validando formulários com prototype de uma forma realmente simples

18 de Junho de 2007

Andrew Tetlaw publicou em seu site uma biblioteca em javascript para validação de formulários usando prototype.

De uma forma realmente simples, esta biblioteca verifica se o campo foi preenchido, se o valor é numérico, se é um e-mail válido, e mais uma série de outras validações.
