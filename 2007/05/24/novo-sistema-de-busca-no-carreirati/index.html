Novo sistema de busca no carreiraTI

24 de Maio de 2007

 

O sistema de busca do carreiraTI foi aprimorado, anteriormente não era possível realizar uma busca por cidade por exemplo. Agora você pode procurar por palavras chaves mais o nome da cidade onde você quer trabalhar (C# São Paulo, por exemplo).

O sistema de busca utilizado é bem simples e baseado no TextSearch, um padrão para busca em Rails. Inicialmente tentei utilizar o plugin Active Search, mas seria necessário alterar muita coisa para funcionar do jeito que eu pretendia, além dele apresentar um erro na hora da gravação do registro.

O carreiraTI está indo muito bem obrigado, mas ainda precisa de mais divulgação. Nunca pensei como seria difícil esta parte. Criar o software foi fácil, divulgar e o tornar conhecido é o grande problema.
