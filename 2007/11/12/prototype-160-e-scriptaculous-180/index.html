Prototype 1.6.0 e script.aculo.us 1.8.0

12 de Novembro de 2007

Novas versões das bibliotecas JavaScript que vêm com o Rails, Prototype 1.6.0 e script.aculo.us 1.8.0, foram lançadas. Já fazia dois anos que o Prototype não recebia uma atualização significativa.

Esta nova versão do Prototype traz uma série de importantes melhorias nas suas APIs, como também corrige uma série de bugs das versões anteriores. Para ver uma lista com algumas das novidades, acesse o blog oficial.

Só para animar um pouquinho, agora o script.aculo.us tem até um nova API para efeitos de som em MP3.
